Spot runner
===========

Run stuff on AWS spot instances easily.


Usage
-----

```shell
$ pip install https://github.com/leadhub-code/spot-runner/archive/master.zip
$ spot-runner -v run_spot_instance --blueprint examples/hello/blueprint.yaml
```
